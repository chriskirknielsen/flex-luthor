Prefer to Wrap Cell Before Content

By default, the content inside of a cell will wrap before the cell will wrap to a new line. To swap this so that the cell will wrap before the content wraps, add the flex-cell-wrap.

First cell
This cell will wrap to a new line before wrapping its content
Third cell
Fourth cell
Fifth cell
View Source 
<div class="fl fl-debug-outline" style="--fl-margin-h: 1em; --fl-margin-v: .5em;">
	<div class="fl-cell">First cell</div>
	<div class="fl-cell fl-cell-wrap">This cell will wrap to a new line before wrapping its content</div>
	<div class="fl-cell">Third cell</div>
	<div class="fl-cell">Fourth cell</div>
	<div class="fl-cell">Fifth cell</div>
</div>
